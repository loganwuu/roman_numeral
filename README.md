# Roman Numerals

##### A web page that converts number entered by the user into Roman numerals. (9/10/15)

#### By Logan Wu & Kevin Tokheim

## Description

This is an app which allows the user to input a number and converts it the Roman numerals. The number has to be smaller than 4,000 and greater than 0. There should also be less than three of the same character in the output.

## Setup

* Clone the project using the link provided on Github in the Terminal.

## Technologies Used

HTML, CSS, JavaScript, jQuery, Mocha and Chai for testing
